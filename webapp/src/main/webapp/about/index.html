  • Swagger UI/

    The Swagger UI for accessing RESTful API of Statements Manager application.
    There are three versions of the Swagger spec files (that drive the UI):
    public (for use by "third-party" apps)
    private (for use by internal apps)
    prototyping (an extended version of the private API). The Swagger spec files are themselves dynamically generated and can be used to generate client-side stubs.

    The RESTful API can return both simple representations and also richer hypermedia representations that are conformant with the Restful Objects spec. The HTTP Accept header is used to select which representation should be generated.
