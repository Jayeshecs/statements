Welcome to Statements Manager, your one stop place for personal finance management. Below are key features of Statements Manager -

  • View and Search personal financial transactions
  • Upload bank account statements (currently HDFC delimited format is supported)
  • Upload credit card statements (currently HDFC PDF format is supported)
  • For each transaction, capability to identify Statement source, Category, Sub-category
  • Deep-learning for automatic categorization of transactions
  • Analytics e.g. Income vs Expense analysis, Unexpected expenses, Financial goals, etc.
Outil™ is a service provider for many commonly used tools such as Statements Manager.
